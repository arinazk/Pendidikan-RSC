AKSANBOOK merupakan sebuah phonebook yang mampu menyimpan informasi kontak, meliputi informasi nama, alamat, dan kekuatan. Program ini dilengkapi dengan beberapa fitur, antara lain:
1. ADD untuk menyimpan kontak baru.
2. DISPLAY untuk menampilkan daftar dari seluruh kontak yang tersedia.
3. SEARCH untuk menampilkan informasi kontak tertentu yang dicari berdasarkan nama.
4. DELETE untuk menghapus kontak tertentu yang dipilih berdasarkan nama.
5. EXIT untuk keluar dari program.
Seluruh fitur dapat digunakandengan mengetik nama fitur menggunakan huruf kapital. Seluruh masukan yang diterima program bersifat case sensitif. Namun, program ini hanya mampu menyimpan maksimal 8 kontak. Ketika program dimulai, program tidak memiliki kontak yang tersimpan. Dan Ketika program ditutup, seluruh informasi yang telah disimpan akan terhapus.
